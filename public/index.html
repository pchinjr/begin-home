Begin apps and APIs

A Begin app is everything you need to build a web application or API using AWS

  • Unified access and logs in the Begin Console
  • Unified tooling for an end to end workflow with OpenJS Architect
  • Cloud database and session management with Begin Data
  • CI/CD, with GitHub integration by default
  • Staging and production environments by default
  • Global CDN, edge caching, static hosting, HTTP endpoints, and more
  • True on demand resources and pay as you go services
  • Portable and transparent code
    • Uses CloudFormation templates to deploy to Begin.com or your own managed AWS account.

Begin is for Developers

Harnesses the power of serverless technology to scale your own capabilities. The technology has been proven to scale, we're here to scale you. Begin removes complexity and curates services tailored to web development. We're big fans of the web!

Begin takes care of:

  • Least priviledge secuity policies - no more over-permissioned IAM roles
  • Automated service discovery - relief from copy/pasting ARNs and juggling endpoints that feel like filing out a mortgage application
  • End to end workflow - Don't worry about local development, build minutes, container images, or any infrastructure configuration
  • AWS services without an AWS account! Don't worry, you can totally add one later
  • Hard limits on spending to prevent a DDOS of your wallet

How does Begin do it?

Begin gives developers additional guide rails for building and deploying serverless applications and APIs on AWS

Open Source framework and convenient libraries to interact with complex architectures

  • OpenJS Architect - Your open source serverless toolbelt
  • Local and offline development server that includes an in-memory database
  • Unopinionated about front-end frameworks and test suites
  • Organized serverless functions, code sharing, and dependencies
  • A workflow based on version control, your infrastructure and your code in the same repo for maximum determinism
  • Outputs standard CloudFormation templates that are compatible with AWS Serverless Application Model (SAM)
